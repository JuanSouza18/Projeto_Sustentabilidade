# Projeto de Sustentabilidade em Python + MySQL

Este projeto foi desenvolvido com o objetivo de praticar a integração entre Python e MySQL através da criação de um sistema voltado para o monitoramento e gerenciamento de recursos naturais, como água, energia e resíduos.

A aplicação permite registrar, armazenar e consultar dados relacionados ao consumo de recursos, auxiliando na análise e conscientização sobre práticas mais sustentáveis.

## Tecnologias Utilizadas

* Python
* MySQL
* SQL
* Bibliotecas de conexão com banco de dados (MySQL Connector)

## Funcionalidades

* Cadastro de recursos monitorados.
* Registro de consumo ou geração de resíduos.
* Armazenamento de informações em banco de dados MySQL.
* Consulta e visualização de dados cadastrados.
* Organização e gerenciamento das informações de forma estruturada.

## Objetivo do Projeto

Este projeto foi criado exclusivamente para fins de estudo e prática, com foco no desenvolvimento de habilidades relacionadas a:

* Programação em Python.
* Manipulação de bancos de dados relacionais.
* Criação e execução de consultas SQL.
* Integração entre aplicações e banco de dados.
* Estruturação de sistemas para gerenciamento de informações.
* Aplicação da tecnologia em soluções voltadas à sustentabilidade.

## Aprendizados

Durante o desenvolvimento deste projeto, foi possível aprimorar conhecimentos sobre persistência de dados, modelagem de banco de dados e comunicação entre Python e MySQL. Além disso, o projeto proporcionou uma visão prática de como a tecnologia pode ser utilizada para auxiliar no monitoramento e na gestão sustentável de recursos.

## Como Executar

1. Clone ou baixe o repositório.
2. Instale as dependências necessárias do Python.
3. Configure o banco de dados MySQL.
4. Execute os scripts de criação das tabelas.
5. Inicie a aplicação Python.
